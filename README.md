# lemon
C语言开发的工具库，包括常用的字符串解析、数据结构、日志库、异步IO线程等

# 开发环境
* gcc version 4.8.3 20140911 (Red Hat 4.8.3-9) (GCC)
* C99
* python

# 本地编译
```
> git clone https://github.com/red-chen/lemon.git --recursive
> cd lemon
> scons
```
